Definición Estilos internos y Externos Selectores Css
lenguaje CSS describe cómo se mostrarán los elementos HTML en la pantalla. El lenguaje CSS ahorra mucho trabajo, ya que permite controlar el diseño de varias páginas web a la vez.
Contiene un conjunto más amplio de tecnologías que permite a los sitios Web y a las aplicaciones ser más diversas y de gran alcance.

Externos Una vez creado, basta incluir un link a ese archivo .css en cada página que queramos aplicar estos estilos. De esta forma, una misma hoja de estilos puede aplicarse a cualquier número de páginas web, simplemente incluyendo en todas ellas el link correspondiente..
Selectores css: Los selectores identifican a un elemento dentro de la página Web para luego poder definir sus propiedades. Los distintos tipos de selectores abarcan desde el simple nombre de las etiquetas usadas en HTML ( BODY , P , CODE , TABLE , UL , etc.).
